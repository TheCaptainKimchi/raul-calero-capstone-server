Welcome to the SparkGG API

Learn how to navigate through the API with the steps below

Paths

  • ("/") Default Path

    This is the default path you are on now. This gives a walkthrough of the SparkGG API

  • ("/leaderboard") Leaderboard Path

    This path has both GET and POST to store match data or display the current leaderboard data calculated from the database list.

  • ("/leaderboard/:puuid") Leaderboard/:puuid Path

    This path will get the leaderboard data of a specific player by using their PUUID.

  • ("/login") Login Path

    This path will authenticate user login details and allow access to receiving data from server.

  • ("/register") Register Path

    This path will create a new user account and store the data within the server.

  • ("/profile") Profile Path

    This path will decode token data and send it back.

  • ("/puuid") Puuid Path

    This path will find the puuid of a player from the Riot Games API

  • ("/matchId") MatchId Path

    This path will find the list of matches of a player using their PUUID.

  • ("/match") Match Path

    This path will find the match details of a player's game using a matchId
